# Kaffiene App Wireframes

## Design System Foundation
- **Typography**: SF Pro (iOS native), Dynamic Type support
- **Colors**: Warm browns (#8B6F47), soft creams (#F5F1E8), system colors
- **Spacing**: 16pt base unit, generous whitespace
- **Components**: Native iOS patterns (Navigation bars, Sheets, Action sheets)
- **Touch Targets**: Minimum 44pt height

## Responsive Considerations

**iPhone SE / Small Screens**:
- Stack cards vertically (no side-by-side)
- Reduce icon sizes slightly
- Tighter spacing (12pt instead of 16pt)

**iPad / Large Screens**:
- Side-by-side layouts where appropriate
- Larger cards, more whitespace
- Multi-column where beneficial

**Dark Mode**:
- Warm browns adapt to dark background
- Maintain contrast ratios
- System colors adapt automatically
